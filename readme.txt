== Description ==
Simple Google Tag Manager is a user-friendly WordPress plugin that simplifies the integration of Google Tag Manager (GTM) into your website. With this plugin, you can effortlessly add both the GTM script and iframe snippet to your WordPress site, enabling advanced tracking, analytics, and marketing functionalities without complex setup processes.

== Installation ==
1. Upload the `google-tag-manager-simplified` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the \'Plugins\' menu in WordPress.
3. Navigate to the \'Google Tag Manager\' settings page in the WordPress admin area.
4. Paste your Google Tag Manager script and iframe snippet into the respective fields.
5. Save your changes and enjoy the enhanced tracking capabilities of Google Tag Manager!

== Frequently Asked Questions ==
To use this plugin, you need to create a Google Tag Manager account and obtain the GTM script and iframe snippet from the GTM dashboard.
